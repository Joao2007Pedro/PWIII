# 🌐 Exercícios de PW III

Repositório com exercícios e pequenos projetos da disciplina **PW III (Programação Web)** do **3º ano do curso de Desenvolvimento de Sistemas da ETEC**.

Os projetos foram desenvolvidos durante as aulas com foco em **lógica de programação aplicada à web e integração com APIs**.

---

## 🛠 Tecnologias Utilizadas

* **JavaScript**
* **Node.js**
* **HTML**
* **APIs externas**

## 📚 Projetos

### 📮 Buscar-CEP

Projeto simples criado para **testar o consumo de uma API de CEP**, permitindo buscar informações de endereço a partir de um CEP informado.

Conceitos praticados:

* Consumo de API
* Requisições HTTP
* Manipulação de dados retornados

---

### 🧮 Calculadora

Aplicação simples de **calculadora**, utilizada para praticar lógica de programação e manipulação de operações matemáticas.

---

### 🧮 Calculadora 2.0

Versão alternativa ou evolução da calculadora anterior, possivelmente com melhorias ou alterações na implementação.

---

### ⚽ Campeonato Paulista

Projeto focado em implementar a **lógica de um campeonato de futebol**, inspirado no **Campeonato Paulista**.

Conceitos praticados:

* Lógica de programação
* Estruturas condicionais
* Organização de dados e resultados
